    init

    init
    (
    • dom
    • [config]
    )

    Defined in develop\main.js:38

    初始化Qtopo实例,创建并将其保存在QTopo.instance数组内,实例api参考QTopo.instance

    Parameters:

    nametypeflagdescription
    dom Document

    指定初始化所在的dom,若是数组则自动取第一个

    [config] Object optional

    配置参数,配置内容为图层scene配置(参考scene配置)

    Returns:

    [instance] QTopo实例对象，QTopo.instance

    Example:

     IPOSS = QTopo.init(
      document.getElementsByClassName("topo_base")[0],
     {
         backgroundColor: "#06243e",//基本属性配置背景色
         path:[] //自定义属性,插入到图层对象scene的额外属性中
     }
 );

    初始化组件模块,加载组件包后可用

    Parameters:

    nametypeflagdescription
    instance Instance

    QTopo生成的实例对象

    config Object

    配置参数

    nametypeflagdescription
    [hideMenu] Boolean optional

    不载入右键菜单

    [hideToolbar] Boolean optional

    不载入工具条

    [hideWindows] Boolean optional

    则不载入窗口模块

    rightMenu Object

    右键菜单初始化参数

    nametypeflagdescription
    filter Array
    过滤掉的菜单的集合
    windows Object

    窗口管理初始化参数

    nametypeflagdescription
    filter Array
    过滤掉的窗口的集合
    toolbar Object

    工具条初始化参数

    nametypeflagdescription
    hideDefaultSearch Boolean
    不加载默认搜索功能

    Returns:

    [Instance]

    实例对象，支持链式调用

    Example:

     IPOSS = QTopo.init(dom, {
                   backgroundColor: "#06243e"
           });
 QTopo.initComponent(IPOSS,
                   {
                       rightMenu: {
                           filter: ["创建节点", "添加链路", "删除", "编辑", "元素切换", "分组操作"]
                       },
                       toolBar: {
                           hideDefaultSearch: true
                       },
                       windows:{
                           filter:["link","imageNode"]
                       }
                   }
           )

    constant

    Object

    Defined in develop\constant.js:8

    定义的通用常量,一般用getType()来区分元素基本类型,getUseType()区分元素具体类型

    Example:

     if(node.getType()==QTopo.constant.NODE){.....}
 if(node.getUseType()!=QTopo.constant.CASUAL){.....}
 if(node.getUseType()==QTopo.constant.node.IMAGE){.....}

    Sub-properties:

    • SCENE String
    • CONTAINER String
    • LINK String
    • LINE String
    • NODE String
    • CASUAL String

      临时元素，用以元素变换收缩等操作的过渡元素,一般不对其做操作

    • node.IMAGE String
    • node.TEXT String
    • container.GROUP String
    • link.CURVE String
    • link.DIRECT String
    • link.FLEXIONAL String
    • link.FOLD String
    • line.DIRECT String
    • mode.NORMAL String
    • mode.EDIT String
    • mode.DRAG String
    • mode.SELECT String
